How to use

  1. On the Dataset page, load a sample dataset, generate synthetic data, or upload your own CSV.
  2. Use Features to inspect the derived features built from raw sensors.
  3. Open Train to fit and compare 7 classifiers.
  4. Use Predict for fault predictions and Explain for SHAP explanations.
  5. Download CSV / HTML / PDF reports from the Report page.

For REST API endpoints see /docs (OpenAPI / Swagger).
